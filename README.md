# BIFS613GroupProject
Biostatistics fall 2019 group project

## R version 3.5.1 (2018-07-02)
## Platform: x86_64-w64-mingw32/x64 (64-bit)
## Running under: Windows 10 x64 (build 18362)

Using R our team performed 4 different statistical Analyses (Descriptive sample summary, Distribution, Hierarchical Clustering, PCA) on TCGA-KICH gene expression data to better understand how gene expression level changes during the different tumor stages.


This repository stores the code and raw data needed to replicate the findings of this team.
